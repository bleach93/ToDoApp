# To Do App – iOS Productivity Application

To Do App is a modern productivity application built with SwiftUI for iOS. The app helps users organize daily tasks with a clean interface, task priorities, reminders, categories, progress tracking, and local persistence.

## Features

* Create, edit, and delete tasks
* Task status workflow:

  * Not Started
  * In Progress
  * Completed
* Priority system:

  * Low
  * Medium
  * High
* Categories:

  * School
  * Work
  * Personal
* Local notifications and reminders
* Repeating reminders:

  * Daily
  * Weekly
  * Monthly
* Search and filter tasks
* Sort tasks by:

  * Priority
  * Reminder date
  * Status
  * Creation date
* Multiple task selection
* Progress tracking with animated progress bar
* Smooth animations and transitions
* Dark mode UI
* Persistent local storage using SwiftData
* Haptic feedback for better user experience

## Technologies Used

* Swift
* SwiftUI
* SwiftData
* UserNotifications Framework
* UIKit (Haptic Feedback)

## Purpose of the Project

This project was created to improve iOS development skills using modern Apple frameworks and best practices for UI/UX design, persistence, and notifications.
